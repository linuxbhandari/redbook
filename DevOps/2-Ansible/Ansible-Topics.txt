MOST ASKED INTERVIEW QUESTIONS
1. What is idempotency in Ansible?
	Running playbook multiple times gives same result without unwanted changes.

2. Difference between command and shell module?
	command → no shell processing
	shell → supports pipes, redirects

3. What are roles?
	Reusable structured automation components.

4. What is Ansible Vault?
	Used to encrypt sensitive data.

5. Why is Ansible agentless?
	Uses SSH; no agent installation required.
