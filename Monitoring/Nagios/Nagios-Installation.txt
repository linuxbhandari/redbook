Nagios Installation Architecture
================================
Nagios Server
   ↓
Plugins
   ↓
Monitored Linux Servers (NRPE)

Service Archtecture
====================
Nagios Server
   ├── Nagios
   ├── check_nrpe plugin
   └── nrpe service

Remote Server
   └── nrpe service

Add remote host monitoring (NRPE)
================================
Install NRPE on target server:

STEP 1: Install EPEL Repository
   # sudo dnf install epel-release -y
   # sudo dnf update -y

STEP 2: Install NRPE and Plugins
   # sudo dnf install nrpe nagios-plugins-all -y


STEP 3: Configure NRPE
   Open config:
   # sudo vi /etc/nagios/nrpe.cfg

Allow Nagios Server IP

Find:
   allowed_hosts=127.0.0.1

Change:
   allowed_hosts=127.0.0.1,<nagios-server-ip>

Example:
   allowed_hosts=127.0.0.1,192.168.1.10


STEP 4: Enable NRPE Service
   # sudo systemctl enable nrpe
   # sudo systemctl restart nrpe
   
   Check status:
   # sudo systemctl status nrpe

STEP 5: Open Firewall Port
   NRPE uses:
      TCP 5666
   
   Allow it:
   # sudo firewall-cmd --permanent --add-port=5666/tcp
   # sudo firewall-cmd --reload
   
   Verify:
   # sudo firewall-cmd --list-ports
