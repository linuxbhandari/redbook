5. NRPE (MOST ASKED)
What is NRPE?
	Nagios Remote Plugin Executor.

	Used for:
		executing checks on remote Linux servers

Flow:
Nagios Server
   ↓
NRPE Agent on remote host
   ↓
Runs local check
   ↓
Returns output

================
Interview Questions

1. What is NRPE?
	Used to run remote checks on Linux systems.

2. Difference between active and passive checks?
	Active → Nagios initiates
	Passive → External app sends result

3. How does Nagios generate alerts?
	Based on plugin return codes and thresholds.

4. What are plugins?
	Scripts/binaries used to perform checks.

5. What is the difference between host and service?
	Host = machine
	Service = monitored component on machine


Interview-ready answer
	Nagios is an infrastructure monitoring tool used to monitor servers, network devices, and applications. It works using plugins that perform service and host checks. NRPE is used for executing remote checks on Linux servers. Nagios supports alerting, escalation, and both active and passive monitoring.
