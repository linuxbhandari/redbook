Q) What is the difference between an image and a container?
	Docker image is a template used to create containers, while a Docker container is the running instance of that image.

Q) What is a Dockerfile?
	A Dockerfile is a text file that contains instructions to automatically build a Docker image.
		It defines:
			Base image
			Application code
			Dependencies
			Commands to run the application
