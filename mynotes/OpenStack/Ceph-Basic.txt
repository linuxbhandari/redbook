What is Ceph?
	Ceph is a distributed storage platform providing scalable and highly available block, object, and file storage.

What is OSD in Ceph?
	OSD stands for Object Storage Daemon. It stores data and handles replication and recovery operations.

What is a Placement Group(PG)?
	Placement Groups are logical groupings used by Ceph to distribute and manage object data across OSDs.

What Happens if an OSD Goes Down?
	Ceph marks the OSD as down, replication and recovery operations start automatically, and data is redistributed to maintain redundancy.
